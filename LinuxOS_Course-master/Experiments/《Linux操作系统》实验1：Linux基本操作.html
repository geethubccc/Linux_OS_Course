《Linux操作系统》实验1：Linux基本操作

一、实验目的

本次实验主要是让大家熟悉Linux的基本操作环境，了解一些常用的用户操作接口（命令）的使用方式，使用C语言编写一段简单的代码，熟悉gcc的基本使用方法。

二、实验方法

本次实验属于验证型实验，按照实验内容的指导完成所有实验步骤，并记录下实验结果，遇到不懂的问题或是在某一步骤上卡壳，先尝试在搜索引擎上寻找解决方法，积极与老师、同学沟通，务必亲自将实验完成。

三、实验内容

  1. 先在应用列表中启动终端窗口“Terminal”，在CLI模式下了解Linux的目录结构，了解Home目录的作用，练习目录相关命令。
cd
pwd
mkdir code
cd code
ls
  1. 创建文件及vi编辑器。
gedit helloProcess.c

将会启动GUI模式的文本编辑器，在里面输入一段C语言代码。

#include <stdio.h>
int main()
{
printf("Hello, process\n");
return 0;
}
  1. 使用gcc对C语言源文件进行编译和链接。
gcc helloProcess.c -o helloProcess
./helloProcess
Hello, process

四、拓展实验

  1. 通过搜索引擎学习apt命令的用法，使用该命令安装软件”sogou拼音输入法“；
  2. 在termial中启动python/python3，比较一下与在Windows中的使用区别，如果该程序包没有安装请使用apt进行下载安装；
  3. 自行完成其他命令的练习，务必要熟练和习惯。
